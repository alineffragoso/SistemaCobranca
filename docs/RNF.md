# Requisitos Não Funcionais

Os requisitos não funcionais definem como o sistema deve se comportar.

## RNF01 - Acesso Web
O sistema deve ser acessado através de navegador web.

## RNF02 - Desempenho
O sistema deve responder às ações do usuário em até 3 segundos.

## RNF03 - Segurança
O sistema deve garantir a proteção dos dados dos clientes.

## RNF04 - Usabilidade
O sistema deve possuir interface simples e fácil de usar.

## RNF05 - Disponibilidade
O sistema deve estar disponível na maior parte do tempo (alta disponibilidade).
