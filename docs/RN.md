# Regras de Negócio

As regras de negócio definem como o processo de cobrança deve funcionar.

## RN01
A segunda notificação só deve ser enviada se não houver resposta da primeira.

## RN02
A negociação só pode ocorrer após contato com o cliente.

## RN03
O cobrador externo só deve ser acionado após falha nas tentativas de contato.

## RN04
O título deve ser protestado caso não haja acordo ou pagamento.

## RN05
O processo de cobrança só deve ser finalizado após pagamento ou devolução do título.

## RN06
Após o pagamento, os valores devem ser transferidos para a empresa contratante.
