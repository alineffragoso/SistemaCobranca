# Requisitos Funcionais

Os requisitos funcionais descrevem as funcionalidades que o sistema deve executar.

## RF01 - Cadastro de Título
O sistema deve permitir cadastrar títulos de cobrança com dados do cliente, valor e vencimento.

## RF02 - Envio de Notificações
O sistema deve enviar automaticamente notificações de cobrança (email, SMS ou WhatsApp).

## RF03 - Registro de Contato
O sistema deve permitir registrar as tentativas de contato telefônico com o cliente.

## RF04 - Negociação de Dívida
O sistema deve permitir registrar negociações de pagamento realizadas com o cliente.

## RF05 - Registro de Pagamento
O sistema deve registrar quando o pagamento for realizado.

## RF06 - Acionamento de Cobrador Externo
O sistema deve encaminhar casos sem retorno para cobrança externa.

## RF07 - Protesto de Título
O sistema deve permitir o envio do título para protesto em caso de não pagamento.
